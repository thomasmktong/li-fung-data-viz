# Admin panel framework based on Angular, Bootstrap 4 and Angular CLI (https://github.com/akveo/ng2-admin)

## How to use
* yarn install
* yarn start

## Libraries included
* TypeScript
* Webpack
* Bootstrap 4 CSS Framework
* Sass
* Angular
* jQuery
* Charts (Chartist, Chart.js)
* Maps (Google, Leaflet, amMap)
